## Product Templates

A Product Template is a JSON document which is persisted on the backend database.  It is used to create items that are then stored on the blockchain. 

## View all product templates 

Here is the GraphQL query

~~~~
query {
  allProducts {
    id
    blockchainId
    maxItems
    name
    publicProduct
    componentName
    publicProduct
    timeCreated
    timeModified
    data
  }  
}
~~~~
